# Inklee

Booking requests without the DM chaos. A lightweight tool for freelance and traveling tattoo artists.

## Docs

- [CLAUDE.md](./CLAUDE.md) — working rules, design system, stack constraints
- [DECISIONS.md](./DECISIONS.md) — product decisions log
- [SLICES.md](./SLICES.md) — build plan

## Dev

```bash
pnpm dev       # start dev server at localhost:3000
pnpm typecheck # type check
pnpm lint      # lint
pnpm build     # production build
```
