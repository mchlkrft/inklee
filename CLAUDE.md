## SEO source of truth

Before performing any work involving SEO, landing pages, metadata, page titles, descriptions, canonicals, redirects, sitemap configuration, structured data, internal linking, indexation, keyword targeting, or SEO content architecture, read:

`docs/seo/inklee-seo-strategy.md`

Responsibility split:

- ChatGPT owns keywords and SEO strategy.
- Claude Code owns implementation and hands-on optimization.

Do not silently redefine keyword ownership or create a new indexable page when the intent already has an owner URL.

When implementation evidence suggests a strategic change, document it as a proposal rather than treating it as approved strategy.

After completing SEO implementation, update:

`docs/seo/seo-implementation-log.md`
